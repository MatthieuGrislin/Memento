Quand t'es dans la merde regarde ici

  1. Lier une feuille CSS
  2. Afficher une image
  3. Pour un tableau

  5. Pour les Div
  6. Pour les formulaires

Pour lier une feuille CSS a un fichier HTML

< link rel="stylesheet" style type="text/css" href="style.css" />

Pour afficher une image

< img src="Images/Afficer une image.png" >

Title pour ajouter un titre. atl pour un texte alternatif si l'image ne s'affiche pas.

Pour un tableau

Thead pour l'entête, Tbody pour le corps et Tfoot pour le pied du tableau. Caption pour le titre. Colspan et Rolspan servent a fusionner des cellules.

Pour un lien hypertext

Title pour mettre un titre au survol.

Pour les Div

width pour la largeur, height pour la hauteur. Float permet d'aligner tant qu'il y a de la place.

Pour les formulaires

Legend = titre, ol = liste a puce ordonnée, label = légende, input = permet a l'utilisateur de saisir les informations.
